The Fab Four technique to create Responsive Emails without Media Queries

Instead of using media query to switch between mobile / desktop layout, here's a trick that uses calc, min-width, max-width and width

<style>
  .item {
    display: inline-block;
    min-width: 50%;
  }
  @media(min-width: 720px) {
    .item {
      width: 25%;
    }
  }
</style>	

<div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
</div>
<style>
  .item {
    display: inline-block;
    min-width: 25%;
    max-width: 50%;
    width: calc((720px - 100%) * 720);
  }
</style>	

<div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
  <div class="item"></div>
</div>
